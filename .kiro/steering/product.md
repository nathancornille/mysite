# Product Overview

This is a personal academic website for Nathan Cornille, built as a static site using Hugo. The site serves as:

- An online portfolio showcasing research work and publications
- A blog for sharing research insights and paper summaries
- A professional presence with CV and contact information

The site is deployed to GitHub Pages at https://nathancornille.github.io/

## Key Features

- Academic profile with photo and bio
- Blog posts about research papers (e.g., COLM 2024 paper on language modeling)
- CV section with embedded PDF viewer
- Contact information
- Responsive design with the Academia Hugo theme

## Target Audience

Academic peers, potential collaborators, and anyone interested in Nathan's research in natural language processing and language modeling.
